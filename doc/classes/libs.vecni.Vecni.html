use_less()

use_less()

Enable LESS CSS plugin, a functional CSS compiler.

Enabling this plugin will generate a new compilied css from the list of less and/or css files defined in style.less file located in app/static/src/css/less/ to style.css file located in app/static/src/gen/css. Note: Ensure that app/static/src/gen folder have the correct permission of 775. This plugin is location in the app/plugins folder.
